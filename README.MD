#Installation:

This command assumes Ubuntu and Gnome are being used. Install dependencies, clone the repo, and install plugins:

    bash -c 'sudo apt-get install vim-nox
    git clone https://github.com/matthewjberger/DotVim.git ~/.vim
    cd ~/.vim
    ./install.sh'
